# 🔍 Data Analysis

> Your data is clean. Now let's actually learn something from it.

---

Data analysis is where cleaned datasets are queried and explored to extract meaningful insights. This can be as simple as a SQL query to answer a business question, or as complex as running machine learning workloads across billions of rows.

## Tools

| Tool | Vendor | Best For | Deployment |
|---|---|---|---|
| Google BigQuery | Google Cloud | Serverless SQL analytics on massive datasets | Cloud |
| AWS Athena | AWS | Querying data directly in S3 using SQL | Cloud |
| Azure Synapse Analytics | Microsoft Azure | Integrated SQL and Spark-based analysis | Cloud |
| Apache Spark | Apache | Distributed in-memory analytics and ML workloads | On-Premise / Cloud |
| Apache Hive | Apache | SQL-based querying on Hadoop data lakes | On-Premise / Self-Hosted |
| Presto / Trino | Open Source | Fast distributed SQL across multiple data sources | On-Premise / Cloud |
| Databricks | Databricks | Collaborative analytics and ML on large datasets | Cloud / SaaS |
| Snowflake | Snowflake | Cloud-native SQL analytics with multi-cluster scaling | Cloud / SaaS |
| Huawei DLI (Data Lake Insight) | Huawei Cloud | Serverless SQL and Spark analytics | Cloud / Server |
